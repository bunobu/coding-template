2. style.scss でディレクトリごと読み込む

## ディレクトリの読み込み順

### 1. global

全体で使用する変数や,mixin,関数などのファイル

### 2. base

サイト共通のスタイルファイル

### 3. plugin

外部ライブラリ CSS ファイルの読み込み

### 4. component

サイト共通の最小単位のパーツ（ボタンなど）

### 5. modules

サイト共通で使いまわしするパーツ（component より大きい）

### 6. page

ページ毎のスタイルファイル
